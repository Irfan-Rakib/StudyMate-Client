## ‚ú® Project Overview

**StudyMate** is a MERN Stack web platform designed to **revolutionize student collaboration** by making it easy to find and connect with the perfect study partners.

In today's learning environment, collaboration is key. StudyMate addresses the challenge of finding compatible peers by allowing users to search and connect based on shared **subjects**, **learning preferences** (e.g., visual, hands-on, or discussion-based), or **nearby locations**. The goal is to make education more interactive, engaging, and goal-oriented.

## üöÄ Features

StudyMate offers a suite of features designed to facilitate seamless student collaboration:

- **User Authentication:** Secure sign-up and login via **Firebase**, supporting both email/password and social login (e.g., Google).
- **Comprehensive Profiles:** Users can create detailed profiles including their subjects, academic level, learning style, and location.
- **Intelligent Partner Search:** Users can filter and search for study partners based on:
  - **Subject:** Math, Physics, History, etc.
  - **Learning Preference:** Discussion-based, problem-solving, test prep.
  - **Location:** Find partners in the same university or neighborhood.
- **CRUD Operations:** Full **C**reate, **R**ead, **U**pdate, and **D**elete functionality for managing user profiles, study group creation, and collaboration posts.
- **Responsive UI:** A user-friendly, responsive design ensures a great experience on any device (mobile, tablet, or desktop).
- **Connection System:** Ability to send connection requests and form dedicated study partnerships.
